Fundamentos da Programação Orientada à Objetos

Essa unidade curricular visa propiciar a aquisição de capacidades básicas relativas à lógica de programação, como ferramenta para desenvolvimento de aplicações, utilizando linguagem orientada a objetos, bem como o desenvolvimento das capacidades socioemocionais adequadas a diferentes situações profissionais.

Carga horária: 175 horas

Professor: Átila Olivi

Aulas às terças-feiras e quintas-feiras

A linguagem de programação usada nas aulas foi Python

Temas abordados nas aulas

  • Lógica de Programação
  • Sintaxe
  • Fluxogramas e pseudocódigo
  • Criação de variáveis
  • Tipos de dados
  • Operadores aritméticos e lógicos
  • Valores booleanos
  • Listas e arrays
  • If/Else (estrutura de decisão)
  • Loops While/For (estruturas de repetição)
  • Funções
  • Classes e objetos
  • Orientação a objetos(Abstração, Polimorfismo, Encapsulamento e Herança)

Atividades e Cursos solicitados

  • Curso Python Programming Essencials
  • Primeira lista (Lógica, fluxogramas e pseudocódigos)
  • Segunda lista (operações lógicas)
  • Terceira lista (operações com loops)
  • Quarta lista (Arrays)
  • Extras Estruturas de decisão
  • Extras Estruturas de repetição
  • Exercícios POO
  • App POO completo

Código de Programação Orientada a Objeto

A superclasse Pessoa define atributos e métodos gerais, que podem ser usados por suas subclasses.

A subclasse Estudante herda os atributos e métodos da superclasse, enquanto aproveita da abstração (usar os métodos de forma que sejam adaptados e façam sentido com o seu propósito.)

O arquivo Main consome ambas as classes, criando objetos com base nelas.
